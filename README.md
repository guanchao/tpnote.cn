# 拓扑笔记 (TpNote)

一款支持双向链接、素材库与 AI 智能助手的跨平台桌面 Markdown 笔记软件，专为个人知识管理而生。

专注本地、离线优先，无需登录，无需云同步，你的笔记完全属于你自己。

---

## ✨ 特性一览

### 📝 所见即所得 Markdown 编辑

基于 TipTap / ProseMirror 的富文本编辑体验，告别分屏预览——你敲下 `#` 它就是标题，敲下 `` ``` `` 它就是代码块，所写即所见。

- 多级标题、段落、引用块
- 粗体、斜体、删除线、行内代码
- 有序列表、无序列表、可勾选任务列表
- 代码块 — 自动语法高亮，覆盖主流编程语言
- 表格 — 支持插入/删除行列、合并单元格、列宽拖拽调整
- 分割线、水平线
- 图片 — 粘贴即保存、拖拽调整大小

### 🔗 双向链接

拓扑笔记的灵魂功能。在任意位置输入 `[[`，立即弹出笔记联想列表，选中即建立双向链接。

- **正向链接**：点击 `[[WikiLink]]` 即刻跳转目标笔记
- **反向链接**：编辑器底部自动聚合所有引用当前笔记的来源及上下文
- **别名支持**：`[[标题|显示别名]]` 语法，链接与显示文本分离
- **智能创建**：链接到不存在的笔记时自动新建，思路不中断

### 🏷️ 标签系统

在笔记中使用 `#标签名` 语法即可创建标签，标签面板自动聚合所有标签及关联笔记，提供除链接之外的另一种知识组织维度。

### 🗺️ 知识图谱

基于 D3.js 力导向图的交互式知识网络可视化。以节点和连线的形式直观展现笔记间的关联关系，支持缩放、拖拽、点击节点跳转至对应笔记。

### 🎨 白板

独立的 `.board` 文件格式，提供自由画布式的可视化思维工具：

- **多类型节点** — 支持文本、图片、文件引用、URL 链接四种节点
- **连线关系** — 节点间可建立有向连接，贝塞尔曲线渲染
- **自动布局** — 基于 Sugiyama 层级布局算法，一键整理节点位置
- **自由操作** — 拖拽移动、缩放画布、调整节点大小、颜色选择
- **导出图片** — 将白板导出为 PNG 图片

### 📐 数学公式

基于 KaTeX 的数学公式渲染：

- **行内公式** — `$E=mc^2$` 语法，行内实时渲染
- **块级公式** — `$$` + 空格触发公式编辑器，支持多行公式
- **双击编辑** — 点击已有公式可随时修改

### 📊 Mermaid 图表

在代码块中使用 `mermaid` 语言标记，即可渲染流程图、时序图、甘特图等多种图表，支持实时预览。

### 🔍 全文搜索

基于 SQLite FTS5 全文检索引擎，输入关键词即刻返回结果，搜索结果附带上下文片段高亮，毫秒级响应。

### 📑 多标签页

支持同时打开多篇笔记，标签页状态持久化，重启后自动恢复：

- 标签页右键菜单：关闭 / 关闭其他 / 关闭全部
- 未保存文件状态指示（圆点标记）
- 关闭未保存文件时弹出确认对话框
- 鼠标中键快速关闭标签页

### 📤 多格式导出

- **PDF** — 基于 Electron printToPDF，高质量页面导出
- **HTML** — 生成独立 HTML 文件，内联样式，无需额外依赖即可在浏览器中打开
- **DOCX** — 基于 docx 库，完整保留标题、列表、表格、代码块等格式

### 🔄 源码模式

一键切换 Markdown 源码编辑模式，直接编辑原始 Markdown 文本，适合需要精确控制格式的场景。支持 Markdown ↔ HTML 双向转换（Turndown / marked）。

### 📸 快照/版本历史

手动创建笔记快照，随时回溯和恢复历史版本：

- **主动快照** — 点击工具栏保存按钮即可创建当前内容的快照，不会自动产生，完全由用户控制
- **快照历史面板** — 查看当前笔记的所有历史快照，按时间倒序排列
- **预览快照** — 点击预览按钮可临时查看历史版本内容，不影响当前文件
- **一键恢复** — 将笔记内容恢复到某个历史快照的状态
- **删除管理** — 可删除不需要的快照，笔记删除时自动清理关联快照

### 📁 文件管理

功能完善的侧边栏文件树：

- 新建笔记（`.md`）、白板（`.board`）、文件夹
- 文件重命名、删除（带确认对话框）
- 文件 / 文件夹拖拽排序
- 多选操作、复制 / 粘贴
- 文件系统实时监听（chokidar），外部修改自动刷新

### 📖 大纲面板

自动提取当前文档的标题层级，生成可点击跳转的文档大纲，支持富文本模式和源码模式。

### 📄 多格式文件支持

不仅仅是 Markdown：

- **文本文件** — `.txt`、`.json`、`.yml`、`.xml` 等以代码编辑器模式打开，支持编辑和保存
- **图片预览** — `.png`、`.jpg`、`.gif`、`.svg` 等图片格式直接预览
- **PDF 预览** — `.pdf` 文件内嵌预览
- **二进制文件** — 自动识别并提示，避免误操作
- **外部文件** — 支持通过系统「打开方式」打开外部文件

### ⚙️ 个性化设置

- **编辑器** — 字体大小、行高、字体族自定义
- **外观** — 亮色 / 暗色主题切换，支持跟随系统偏好
- **保存** — 自动保存开关及延迟时间自定义
- **快捷键** — 查看所有可用快捷键

### 🌗 亮暗主题

精心设计的亮色与暗色主题，支持手动切换或自动跟随系统偏好。

### 🤖 AI 智能助手

集成 DeepSeek、智谱 GLM、通义千问、月之暗面等多家主流大模型，提供两种 AI 交互模式：

- **AI 对话模式** — 侧边栏 AI 面板，基于当前文章上下文自由问答、讨论和获取灵感
- **AI Agent 模式** — AI 自主阅读、编辑笔记，自动执行读取、插入、替换、生成图片等多步操作
- **AI 文生图** — 基于大模型的文本生成图片能力，选中文字描述或通过 Agent 指令即可生成配图
- **AI 摘要** — 一键生成文章摘要，快速把握核心内容
- **AI 标签推荐** — 智能推荐标签，自动归类知识
- **AI 内容润色** — 帮你优化文字表达，让笔记更清晰流畅

### 📡 素材库

内置 RSS 订阅管理器、关键字搜索引擎和网页剪藏工具，让优质信息主动流入你的知识库：

- **RSS 订阅** — 支持 RSS/Atom 格式订阅，自动拉取新文章，定时刷新
- **关键字搜索** — 聚合 Google、Bing、百度三大搜索引擎，按关键字追踪最新资讯
- **网页剪藏** — 输入 URL 一键抓取网页标题和正文，智能编码检测
- **机器人推送** — 新文章自动推送到企业微信、钉钉、飞书、Telegram 群聊机器人
- **素材转笔记** — 一键将素材转化为 Markdown 笔记，自动索引到知识库

## 📖 使用指南

1. **打开笔记库** — 首次启动点击「打开笔记库」，选择一个文件夹作为你的 Vault
2. **新建笔记** — 侧边栏点击「+ 新建笔记」，或右键菜单创建笔记 / 白板 / 文件夹
3. **编辑** — 直接在编辑器中书写，支持完整 Markdown 语法，所见即所得
4. **插入链接** — 输入 `[[` 触发笔记联想，选中后自动插入双向链接
5. **插入标签** — 输入 `#标签名` 创建标签，在标签面板浏览和筛选
6. **查看反向链接** — 编辑器底部展示所有引用当前笔记的来源
7. **知识图谱** — 左侧活动栏点击图谱图标，可视化浏览笔记关联
8. **白板** — 新建 `.board` 文件，自由创建节点和连线，整理思路
9. **数学公式** — 行内公式 `$E=mc^2$`，块级公式 `$$` + 空格后弹窗编辑
10. **Mermaid 图表** — 创建 `mermaid` 代码块，实时渲染图表
11. **导出** — 右键菜单导出当前笔记为 PDF / HTML / DOCX
12. **搜索** — 侧边栏搜索框输入关键词，全文检索
13. **保存** — `⌘S` / `Ctrl+S` 手动保存，或启用自动保存

## 🔒 数据说明

- 笔记以 `.md` 纯文本文件存储在你选择的 Vault 目录中
- 白板以 `.board` JSON 文件存储，与笔记同目录
- `.topo-note/index.db` 是 SQLite 索引文件，仅用于加速搜索和链接查询，可随时从源文件重建
- 图片随笔记存储在本地，通过相对路径引用
- **不会向任何服务器发送数据**
