强大且克制的功能

每一个功能都服务于同一个目标——让你更流畅地记录和思考

🔗

双向链接

输入 [[ 即刻弹出联想列表，建立笔记间的双向引用关系。反向链接面板自动聚合所有引用来源。

📝

所见即所得编辑

基于 TipTap 的富文本编辑器，告别分屏预览。支持标题、列表、代码块、表格、公式等完整 Markdown 语法。

🗺️

知识图谱

基于 D3.js 力导向图的交互式知识网络可视化，以节点和连线直观展现笔记间的关联关系。

🎨

自由白板

独立画布式思维工具，支持文本、图片、连线等多种节点，自动布局算法一键整理。

📐

数学公式

基于 KaTeX 的数学公式渲染，支持行内公式和块级公式，双击即可编辑。

📊

Mermaid 图表

在代码块中使用 mermaid 标记，即可渲染流程图、时序图、甘特图等多种图表。

📤

多格式导出

支持将笔记导出为 PDF、HTML、DOCX 三种格式，完整保留排版和样式。

🏷️

标签系统

使用 #标签名 语法即刻创建标签，标签面板自动聚合关联笔记，多维度组织知识。

📸

快照/版本历史

手动创建笔记快照，随时预览和恢复历史版本。完全由用户主动触发，不自动产生，安心掌控每一次存档。

💬

AI 对话模式

在编辑器侧边栏打开 AI 对话面板，基于当前文章上下文与 AI 自由交流。提问、讨论、获取灵感，AI 理解你的笔记内容。

🤖

AI Agent 模式

AI 自主阅读、编辑你的笔记——自动读取文章、插入内容、替换文本、生成配图。一句指令，Agent 自动完成多步操作。

🖼️

AI 文生图

基于大模型的文本生成图片能力，选中文字描述或通过 Agent 指令即可生成配图，图片自动插入光标位置。

✨

AI 摘要 & 润色

集成多家大模型（DeepSeek、智谱、通义千问等），选中文本一键生成摘要、智能推荐标签、内容润色，让 AI 为你的笔记提效。

📃

块引用

支持多级嵌套块引用，快捷键或工具栏一键操作。适合引用原文、标注重点、记录灵感，让笔记层次分明。

📡

RSS 订阅

内置 RSS/Atom 订阅管理器，添加订阅源后自动拉取新文章。支持定时刷新、批量管理，让优质信息主动流入你的知识库。

📎

网页剪藏

输入任意网页 URL，一键抓取标题和正文内容，智能编码检测、正文提取。剪藏的素材可添加标签，随时转为笔记。

🤖

机器人推送通知

RSS 新文章自动推送到企业微信、钉钉、飞书、Telegram 群聊机器人。多机器人并行推送，重要信息不遗漏。

设计理念

本地优先，数据完全属于你

笔记以 .md 纯文本文件存储在你自己的电脑上。没有账号、没有云端、没有同步冲突。你可以用 Git 管理版本，用任何文本编辑器打开它们。

  • 纯 .md 文件存储，永不被锁定
  • 不向任何服务器发送数据
  • 完全离线可用，无需网络
  • SQLite 索引可随时从源文件重建

  • 完全由用户主动触发，不自动产生

为什么做这个项目

拓扑笔记是我的个人项目。因为觉得市面上的笔记软件总有些不完善的地方——要么过于臃肿，要么功能缺失，要么需要付费使用，要么数据不在自己手里——所以就自己动手做了一个。

它完全免费、开源，希望也能帮到同样在寻找一款趁手笔记工具的你。如果在使用过程中遇到任何问题或有建议，欢迎随时通过邮件联系我，我会认真对待每一条反馈。
